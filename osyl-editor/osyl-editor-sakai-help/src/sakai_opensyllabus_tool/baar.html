Supprimer une référence bibliographique

Vous avez deux scénarios possibles pour supprimer une référence bibliographique.

Dans le premier cas, vous enlevez simplement une référence bibliographique d'une section du plan de cours.  Elle reste, cependant, disponible via votre outil Ressources et pourra éventuellement être insérée à nouveau dans le plan de cours.

Pour ce faire :

  1. Dans le Menu du cours (à gauche) du plan de cours, cliquez sur le nom de la section (ex : Matériel pédagogique) dans laquelle se trouve la référence que vous voulez enlever.
  2. Passez le curseur de votre souris sur le lien que vous souhaitez supprimer dans la zone d'édition (à gauche) et cliquez sur le bouton Supprimer qui apparaît.
  3. Sur l'écran de confirmation, cliquez sur Ok pour supprimer cette référence du plan de cours.

Cependant, si vous souhaitez supprimer une référence bibliographique complètement de la liste de références du cours (elle n'apparaîtra plus dans la liste des références et ne pourra plus être ajoutée dans le plan de cours), voici la procédure :

  1. Cliquez sur l'outil Ressources dans le menu des outils du cours (à gauche).
  2. Dans l'interface qui s'ouvre, il y a un lien intitulé Références bibliographiques du cours.
  3. Cliquez sur Actions à droite du lien Références bibliographiques du cours et dans le menu déroulant qui s'ouvre, sélectionnez l'option Réviser la liste des références.
  4. Dans la liste qui apparaît, se trouvent toutes les références que vous avez déjà ajoutées.
  5. Pour supprimer une référence, cliquez sur le lien enlever, en dessous de chaque référence.
