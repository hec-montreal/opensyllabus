Ajouter, modifier et supprimer un hyperlien

L'enseignant peut ajouter différents types de contenu au plan de cours : textes, documents, hyperliens, etc.  Il est possible d'ajouter ces types de contenus dans plusieurs sections du cours. Les étapes pour ajouter ces contenus sont toujours les mêmes, peu importe la section du plan de cours.

Ajouter un hyperlien

La section ci-dessous montre les étapes pour ajouter un hyperlien dans le plan de cours.

  1. Dans le menu du cours (rectangle de gauche), cliquez sur la section du plan de cours à laquelle vous voulez ajouter un hyperlien (ex : présentation du cours, matériel pédagogique). Note : Il n'est pas possible d'ajouter un hyperlien dans toutes les sections.
  2. Dans l'espace d'édition à droite, cliquez sur Ajouter et dans le menu déroulant, choisissez Hyperlien.
  3. L'élément intitulé Non défini- Tapez votre texte ici s'affiche à l'écran.
  4. Passez le curseur de votre souris sur Tapez votre texte ici' et cliquez sur le bouton Modifier qui apparaît.
  5. Dans la boîte qui s'ouvre, saisissez un hyperlien et son libellé (le texte cliquable qui est affiché au lieu de l'adresse), respectivement dans les cases intitulées URL et Libellé.
  6. Sélectionnez un type de ressource dans le menu déroulant : "Type de ressource".

Le reste des paramètres permettent de définir le contexte d'utilisation de l'hyperlien.

  1. En partant du haut à gauche, la première option s'intitule Rubrique et la seconde Niveau de diffusion. Pour plus d'informations sur les rubriques et sur les niveaux de diffusion, lisez la section de l'aide Rubriques et niveau de diffusion.
    Ensuite, il y a deux Options qui peuvent être activées, en cochant la case à côté de chacune. L'option Caché permet de cacher l'hyperlien suite à la publication, et de le rendre visible au moment opportun. L'option Important permet de visuellement attirer l'attention sur l'hyperlien, avec un encadré bleu.
    Le Niveau d'exigence permet de définir si cet hyperlien est obligatoire, complémentaire ou recommandé.
  2. En dessous de la case pour saisir l'URL, se trouve une zone de texte pour saisir un commentaire au besoin.
  3. En dessous de la zone réservée aux commentaires se trouve l'option Déplacer vers. Celle-ci permet de déplacer l'hyperlien dans une autre section du plan de cours (ex : matériel pédagogique, présentation du cours).
  4. Pour sauvegarder les modifications apportées, cliquez sur le bouton OK, en bas à droite de la fenêtre.

Modifier un hyperlien

Pour modifier un hyperlien dans le plan de cours :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle se trouve l'hyperlien que vous souhaitez modifier (ex : matériel pédagogique).
  2. Passez le curseur de votre souris sur l'hyperlien que vous souhaitez modifier et cliquez sur le bouton Modifier qui apparaît.
  3. Dans l'interface qui s'ouvre, vous êtes en mesure  de saisir/modifier l'hyperlien et définir/modifier ses paramètres (voir ci-dessus pour la procédure complète). 
  4. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite dans l'interface.

Supprimer un hyperlien

Pour supprimer un hyperlien dans le plan de cours :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle se trouve l'hyperlien que vous souhaitez supprimer (ex : matériel pédagogique).    
  2. Passez le curseur de votre souris sur l'hyperlien à supprimer et cliquez sur le bouton Supprimer qui apparaît.
  3. Sur l'écran de confirmation, cliquez sur Ok pour supprimer le contenu.
