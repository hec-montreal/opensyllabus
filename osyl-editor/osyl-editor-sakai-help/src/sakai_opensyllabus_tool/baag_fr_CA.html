Ajouter, modifier et supprimer un document

L'enseignant peut ajouter différents types de contenu au plan de cours : textes, documents, hyperliens, etc.  Il est possible d'ajouter ces types de contenus dans plusieurs sections du plan de cours. Les étapes pour ajouter ces contenus sont toujours les mêmes, peu importe la section où l'on se trouve.

Éléments à prendre en considération

L'enseignant doit d'abord téléverser (en anglais: upload) le document sur le serveur, en passant par par le plan de cours (méthode recommandée) ou par l'outil Ressources. Dans les deux cas, le document est téléversé dans le dossier Ressources. Une fois le fichier téléversé, l'enseignant peut l'insérer dans le plan de cours en créant un lien vers ce document. Il est possible d'avoir plusieurs liens vers le même document à partir de plusieurs sections du plan de cours. Le même document peut, ainsi, se trouver dans le « Matériel pédagogique » et dans les « Évaluations ». La taille du fichier téléversé ne peut excéder 100 Mo. Il n'y a aucune restriction quant au nombre et aux types de fichiers.

Au moment du téléversement, vous devez choisir une option par rapport aux droits d'auteur du document. Il vous sera impossible d'insérer ce document dans le plan de cours, si vous n'avez pas déclaré détenir les droits d'auteur sur ce document ou les avoir obtenus. Si le document est une œuvre publiée (ex : article de périodique), il est fortement recommandé de l'ajouter comme une référence bibliographique, plutôt que de téléverser le fichier, pour ne pas enfreindre les droits d'auteur.

Au moment du téléversement, vous devez aussi choisir le type de ressource, par exemple : article, cas, vidéo, etc. Cette information est très importante notamment pour les étudiants, car elle leur permet de voir d'emblée à quel type de ressource ils ont affaire.

Il est possible de créer des sous-dossiers pour organiser les fichiers à l'intérieur du dossier Ressources.

Si vous déplacez un fichier d'un dossier à l'autre, le lien entre ce fichier et le plan de cours sera brisé et devra être refait.

Ajouter un document

Pour ajouter un document dans le plan de cours, il y a deux étapes principales  :

  1. téléverser le fichier sur le serveur de ZoneCours 2.0 ;
  2. insérer le fichier dans une section du plan de cours.

Il existe deux façons de procéder pour joindre des documents:

  • en passant directement par le plan de cours;
  • en déposant le document dans l'outil Ressources, comme pour des références bibliographiques.

A. Ajouter un document directement dans le plan de cours

Dans le Menu du cours (rectangle de gauche), cliquez sur la section du plan de cours dans laquelle vous voulez ajouter un document (ex : présentation du cours, matériel pédagogique). Note : Il n'est pas possible d'ajouter un document dans toutes les sections.

  1. Dans l'espace d'édition à droite, cliquez sur Ajouter et dans le menu déroulant, choisissez Document.
  2. L'élément intitulé Non défini- Tapez votre texte ici s'affiche à l'écran.
  3. Passez le curseur de votre souris sur Tapez votre texte ici et cliquez sur le bouton Modifier qui apparaît.
  4. Dans la fenêtre qui s'ouvre, vous pouvez téléverser un document, sélectionner un document et définir son contexte d'utilisation. L'interface est composée de deux parties. La partie du haut intitulée Sélectionner un document  qui vous permet de téléverser un document ou de sélectionner un document parmi ceux déjà déposés dans le dossier Ressources du cours. La partie du bas intitulée Contexte d'utilisation vous permet de définir les paramètres et le contexte d'utilisation du document que vous voulez ajouter. Note : Un même document téléchargé dans plusieurs sections du plan de cours peut avoir des contextes d'utilisation différents.
  5. Pour déposer  un document sur le serveur : dans la partie de l'interface intitulée Sélectionner un document, cliquez sur la troisième icône à droite, illustrée par une feuille blanche et un signe + vert. Note : par défaut, le fichier sera téléversé dans le dossier du cours. Si vous voulez créer des sous-dossiers cliquez sur l'icône du milieu (dossier jaune et un signe + vert).
  6. Une fenêtre s'ouvre qui vous permet d'aller le chercher sur votre ordinateur.
    • Vous devez absolument sélectionner une des deux options dans le menu déroulant concernant les droits d'auteur et une option dans le menu déroulant sur le type de ressource. L'application vous permet d'insérer uniquement des documents pour lesquels vous détenez les droits d'auteur ou vous les avez obtenus.
    • Vous devez aussi sélectionner le type de ressource.
  7. Le document apparaît dans le rectangle de la section Sélectionner un document et il est déjà préselectionné.
  8. Les paramètres qui se trouvent dans la partie du bas permettent de définir le contexte d'utilisation.
    • En partant du haut à gauche, la première option s'intitule Rubrique et la seconde Niveau de diffusion. Pour plus d'informations sur les rubriques et sur les niveaux de diffusion, lisez la section de l'aide Rubriques et niveau de diffusion.
    • Ensuite, il y a deux Options qui peuvent être activées, en cochant la case à côté de chacune. L'option Caché permet de cacher le document suite à la publication, et de le rendre visible au moment opportun. L'option Important permet de visuellement attirer l'attention sur le document, avec un encadré bleu.
    • Le Niveau d'exigence permet de définir si le document est obligatoire, complémentaire ou recommandé.
    • Juste en dessous des paramètres mentionnés ci-dessus, se trouve l'unique paramètre obligatoire, le Texte cliquable. Le texte qui est saisi dans ce champ est le lien cliquable qui sera affiché dans le plan de cours pour ouvrir le document.
    • En dessous, se trouve également un champ de texte facultatif de commentaires.
  9. En dessous de la zone de texte, se trouve l'option Déplacer vers. Celle-ci permet de déplacer ce document dans une autre section du plan de cours (ex : matériel pédagogique, présentation du cours).
  10. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite dans la fenêtre.

B. Ajouter un document via l'outil Ressources

  1. Cliquez sur l'outil Ressources dans le menu de gauche du site de cours.
  2. Par défaut, vous voyez le dossier de votre cours nommé selon le numéro de répertoire. Vous pouvez y entrer des documents ou vous pouvez créer un nouveau dossier. Cliquez sur le menu déroulant Ajouter et sélectionner Téléverser fichier(s) ou Nouveau(x) dossier(s).
  3. Cliquez sur le bouton Parcourir..., localisez et sélectionnez le fichier désiré, puis cliquez sur Ouvrir.
  4. Dans la zone de texte « Nom d'affichage », entrez le titre que vous souhaitez donner au fichier dans l'outil Ressources. Si vous laissez cette case vide, le titre d'origine du fichier sera utilisé.
  5. Pour ajouter une description ou tout autre détail (notamment qui a accès au fichier, ou encore le moment à partir duquel il sera visible), cliquez sur Ajouter des détails pour cet élément.
  6. Vous pouvez choisir d'envoyer un avis par courriel à vos étudiants à l'aide du menu déroulant qui se trouve au bas de la page. Choisissez Priorité élevée (tous les participants) pour que tous vos étudiants reçoivent votre avis par courriel. Choisissez Priorité basse (Seulement les étudiants qui en ont fait la demande) pour que ceux qui veulent recevoir tous les avis par courriel soient avisés.
  7. Cliquez sur Ajouter un autre fichier pour téléverser un autre fichier, si vous le souhaitez.
  8. Cliquez sur Téléverser les fichiers.
  9. Cliquez sur Plan de cours dans le menu de gauche du site de cours.
  10. Dans l'espace d'édition à droite, cliquez sur Ajouter et dans le menu déroulant, choisissez Document.
  11. L'élément intitulé Non défini- Tapez votre texte ici s'affiche à l'écran.
  12. Passez le curseur de votre souris sur Tapez votre texte ici et cliquez sur le bouton Modifier qui apparaît.
  13. Dans la fenêtre qui s'ouvre, vous pouvez sélectionner un document et définir son contexte d'utilisation. L'interface est composée de deux parties. La partie du haut intitulée Sélectionner un document qui vous permet de sélectionner un document parmi ceux déjà déposés dans le dossier Ressources. La partie du bas intitulée Contexte d'utilisation vous permet de définir les paramètres et le contexte d'utilisation du document que vous voulez ajouter. Note : Un même document téléchargé dans plusieurs sections du plan de cours peut avoir des contextes d'utilisation différents.
  14. IMPORTANT  : Si vous n'avez pas ajouté de détails précédemment lors du téléchargement dans l'outil ressources, cliquez sur Voir les propriétés du document et :
    1. sélectionnez l'option qui convient concernant les droits d'auteur que vous avez sur ce document;
    2. sélectionnez le type de ressource qui correspond au document;
    3. cliquez sur "Sauvegarder".
  15. Les autres paramètres qui se trouvent dans la partie du bas permettent de définir le contexte d'utilisation. En partant du haut à gauche, la première option s'intitule Rubrique et la seconde Niveau de diffusion. Pour plus d'informations sur les rubriques et sur les niveaux de diffusion, lisez la section de l'aide Rubriques et niveau de diffusion.
    Ensuite, il y a deux Options qui peuvent être activées, en cochant la case à côté de chacune. L'option Caché permet de cacher le document suite à la publication, et de le rendre visible au moment opportun. L'option Important permet de visuellement attirer l'attention sur le document, avec un encadré bleu. Le Niveau d'exigence permet de définir si le document est obligatoire, complémentaire ou recommandé.
    Juste en dessous des paramètres mentionnés ci-dessus, se trouve l'unique paramètre obligatoire, le Texte cliquable. Le texte qui est saisi dans ce champ est le lien cliquable qui sera affiché dans le plan de cours pour ouvrir le document. En dessous, se trouve également un champ de texte facultatif de commentaires.
  16. En dessous de la zone de texte, se trouve l'option Déplacer vers. Celle-ci permet de déplacer ce document dans une autre section du plan de cours (ex : matériel pédagogique, présentation du cours).
  17. Pour sauvegarder les modifications apportées, cliquez sur le bouton OK, en bas à droite dans la fenêtre.
 

Modifier le contexte d'utilisation d'un document

Pour modifier le contexte d'utilisation d'un document dans le plan de cours :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle se trouve le document  (ex : matériel pédagogique).
  2. Passez le curseur de votre souris sur le document dont vous souhaitez modifier le contexte d'utilisation et cliquez sur le bouton Modifier qui apparaît.
  3. Dans l'interface qui s'ouvre, vous pouvez modifier le contexte d'utilisation du document. Voir ci-dessus les étapes 8 et suivantes.
  4. Pour sauvegarder les modifications apportées, cliquez sur le bouton OK, en bas à droite.

Pour changer un document que vous avez modifié sur votre ordinateur ou pour le remplacer, reprenez les étapes décrites dans le paragraphe « Ajouter un document ». Attention, une nouvelle version d'un document n'écrase pas la précédente. Cela veut dire qu'il faut changer le lien si vous voulez remplacer un document dans le plan de cours et détruire l'ancienne version du document pour éviter un confusion éventuelle et une surcharge du serveur.

Supprimer un document

Pour supprimer un document dans le plan de cours :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle se trouve le document que vous voulez supprimer (ex : matériel pédagogique). 
  2. Dans l'espace d'édition à droite, passez le curseur de votre souris au dessus du document que vous souhaitez supprimer et cliquez sur le bouton Supprimer qui apparaît suite à cette action.
  3. Sur l'écran de confirmation, cliquez sur Ok pour supprimer le contenu.
