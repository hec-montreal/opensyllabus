Ajouter une référence bibliographique dans le plan de cours

Les références bibliographiques peuvent seulement être insérées dans les sections Matériel pédagogique, Évaluations et Organisation du cours. Pour l'ajout des références bibliographiques, vous devez être dans l'outil plan de cours, comme pour l'ajout d'autres types de contenus (ex : texte, hyperlien, etc.)

Ajouter une référence bibliographique dans le plan de cours :

  1. Cliquez sur le nom de la section (ex : Matériel pédagogique) auquel vous souhaitez ajouter la référence, dans le Menu du cours (rectangle à gauche),
  2. Dans la zone d'édition (rectangle à droite), cliquez sur Ajouter et dans le menu déroulant qui s'ouvre, sélectionnez Référence bibliographique.
  3. L'élément intitulé Non-défini- Référence bibliographique pas encore sélectionnée apparaît, dans la zone d'édition.
  4. Passez le curseur de votre souris au-dessus de cet élément et cliquez sur le bouton Modifier qui apparaît suite à cette action.
  5. Pour sélectionner la référence bibliographique, double-cliquez sur (LREF) Références bibliographiques, en dessous du titre du cours, pour afficher toutes les références bibliographiques du cours.
  6. Cliquez sur la référence que vous souhaitez ajouter et le titre de celui-ci apparaît dans le champ Réf. Sélectionnée.
NOTE : Attention, vous avez la possibilité de spécifier un type pour la référence bibliographique pour l'utiliser dans le plan de cours. Pour ce faire, cliquez sur la flèche qui pointe vers le bas, à côté de Voir la référence bibliographique. Dans le menu :"Voir la référence bibliographique". Sélectionnez le type de la ressource dans le champs : "type de document", et cliquez sur "Sauvegarder".
  1. Si vous avez créé la référence bibliographique manuellement : N'oubliez pas de désactiver le lien vers la bibliothèque (il est activé par défaut). Sinon, le lien cliquable vers la bibliothèque générera un message d'erreur. Pour désactiver le lien vers la bibliothèque, cliquez sur la petite flèche qui pointe vers le bas, à côté de Voir la référence bibliographique (juste en dessous du champ intitulé Réf. Sélectionné). Trois options avec cases à cocher s'affichent. Cochez la première case, intitulée Désactiver le lien vers la bibliothèque.
  2. Les autres paramètres qui se trouvent dans la partie du bas permettent de définir le contexte d'utilisation.
  3. En partant du haut à gauche, la première option s'intitule Rubrique et la seconde Niveau de diffusion. Pour plus d'informations sur les rubriques et sur les niveaux de diffusion, lisez la section de l'aide Rubriques et niveau de diffusion.
    Ensuite, il y a deux Options qui peuvent être activées, en cochant la case à côté de chacune. L'option Caché permet de cacher la référence suite à la publication, et de le rendre visible au moment opportun. L'option Important permet de visuellement attirer l'attention sur la référence, avec un encadré bleu.
    Le Niveau d'exigence permet de définir si le document/référence est obligatoire, complémentaire ou recommandé.
    En dessous, se trouve également un champ de texte facultatif de commentaires.
  4. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite de la fenêtre.

Note : La même référence bibliographique peut être ajoutée dans plusieurs sections du plan de cours, avec des contextes d'utilisation différents, si souhaité.
