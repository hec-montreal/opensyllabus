Ajouter, modifier et supprimer une évaluation

L'enseignant doit définir les méthodes d'évaluation (ex : travail, examen intra, examen final, etc.) utilisées dans le cadre du cours dans le plan de cours. Les évaluations peuvent uniquement être ajoutées dans la section Évaluations du plan de cours.

Vous devez d'abord ajouter l'activité d'évaluation  (ex : examen final) et son contexte d'utilisation. Par la suite, vous pourrez ajouter du contenu (ex : texte, documents, hyperliens, etc.) lié à cette évaluation.

Ajouter une évaluation

Pour ajouter une activité d'évaluation dans le plan de cours :

  1. Dans la zone à gauche intitulé Menu du cours, cliquez sur la section Évaluations.                                               
  2. Dans la zone d'édition à droite, cliquez sur Ajouter et dans le menu déroulant, choisissez Évaluation. L'élément intitulé Évaluation s'affiche à l'écran, dans le rectangle à droite.
  3. Passez le curseur de votre souris sur cet élément et cliquez sur le bouton Modifier qui apparaît.
  4. Dans l'interface qui s'ouvre, saisissez une méthode d'évaluation et définissez son contexte d'utilisation.
  5. En partant du haut à gauche, la première option s'intitule n°. Celle-ci permet de classer les éléments grâce à une classification supplémentaire (des numéros ou des lettres). Important: Une fois utilisée, il est impossible de revenir à la numérotation par défaut. Pour modifier l'ordre des éléments, utilisez plutôt les petites flèches grises à droiet de chacun des éléments.
  6. Le Type de l'évaluation vous permet de choisir la nature de l'évaluation (ex : examen intra, examen final, travail, etc.).
  7. Pondération permet d'accorder un pourcentage à l'évaluation en question.
  8. Date de l'évaluation permet d'afficher la date à laquelle l'évaluation aura lieu. Note: pour les examens, il est impossible de remplir ce champ. Pour connaître les dates d'examens, l'étudiant doit consulter HEC en ligne.
  9. En dessous de ces paramètres obligatoires, se trouvent des options facultatives pour mieux décrire la méthode d'évaluation. En cochant les cases appropriées, vous êtes en mesure de définir la localisation (en classe / hors classe), la modalité (oral/écrit) et le mode de remise (papier/électronique). Note : Les options ne sont pas exclusives et les deux cases d'un paramètre peuvent être cochées (ex : un travail pourrait être remis en classe et/ou électroniquement).
  10. En dessous de ces options, se trouve le champ pour saisir le Nom de l'évaluation. Par défaut, le nom affiché est le même que le type d'évaluation choisi dans le menu déroulant (ex : «travail» si c'est l'option que vous avez choisie). Cependant, vous pouvez le modifier afin de donner plus d'informations aux étudiants.
  11. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite.

Pour ajouter des contenus à l'intérieur de l'activité d'évaluation :

  1. Cliquez sur le nom de l'élément (ex : examen intra) dans le Menu du cours à gauche.
  2. Dans la zone à droite, cliquez sur Ajouter et choisissez l'élément à ajouter (ex : texte, document, etc.). Vous avez beaucoup de flexibilité pour décrire votre méthode d'évaluation et vous pouvez y ajouter du contenu texte, des documents, des références bibliographiques. Comme les interfaces et paramètres dépendent du (des) type(s) d'élément(s) que vous ajoutez, et que ces éléments ne sont pas spécifiques à la section Évaluations, consultez les sections de l'Aide qui concernent l'ajout de différents types de contenus.

Modifier une évaluation

Pour modifier une évaluation dans le plan de cours :

  1. Dans le rectangle à gauche intitulé Menu du cours, cliquez sur la section Évaluations.
  2. Passez le curseur de votre souris sur l'activité évaluation que vous souhaitez modifier et cliquez sur le bouton Modifier qui apparaît.
  3. Dans l'interface qui s'ouvre, vous pouvez modifier l'évaluation et définir son contexte d'utilisation. Référez-vous au paragraphe précédent de cette page.
  4. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite.

Supprimer une évaluation

Pour supprimer une évaluation dans le plan de cours :

  1. Dans le rectangle à gauche intitulé Menu du cours, cliquez sur la section Évaluations.
  2. Passez le curseur de votre souris dans le rectangle de droite sur l'activité d'évaluation que vous souhaitez supprimer et cliquez sur le bouton Supprimer qui apparaît.
  3. Sur l'écran de confirmation, cliquez sur Ok pour supprimer le contenu.
