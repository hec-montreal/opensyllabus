Ajouter, modifier et supprimer du contenu texte

L'enseignant peut ajouter différents types de contenu au plan de cours : textes, documents, hyperliens, etc.  Il est possible d'ajouter ces types de contenus dans plusieurs sections du cours. Les étapes pour ajouter ces contenus sont toujours les mêmes, peu importe la section du plan de cours.

Ajouter du texte

Marche à suivre pour ajouter du contenu texte dans le plan de cours.

  1. Dans le Menu du cours (rectangle de gauche), cliquez sur la section du plan de cours à laquelle vous voulez ajouter du contenu texte (ex : présentation du cours, matériel pédagogique). Note : Il n'est pas possible d'ajouter du contenu texte dans toutes les sections.
  2. Dans l'espace d'édition à droite, cliquez sur Ajouter et dans le menu déroulant, choisissez Texte.
  3. L'élément intitulé Non défini- Tapez votre texte ici s'affiche à l'écran.
  4. Passez le curseur de votre souris sur Tapez votre texte ici et cliquez sur le bouton Modifier qui apparaît.
  5. Dans la boîte de texte qui s'ouvre, saisissez un texte et définissez des paramètres et les contextes d'utilisation de ce bloc de texte. Les fonctionnalités de mise en page se trouvent juste au dessus de cette zone. En partant du haut à gauche, la première option s'intitule Rubrique et la seconde Niveau de diffusion. Pour plus d'informations sur les rubriques et sur les niveaux de diffusion, lisez la section de l'aide Rubriques et niveau de diffusion.
    Ensuite, il y a deux Options qui peuvent être activées, en cochant la case à côté de chacune. L'option Caché permet de cacher le bloc de texte suite à la publication, et de le rendre visible au moment opportun. Cette fonctionnalité peut être utile pour rédiger un brouillon, une information qui reste à vérifier, etc. L'option Important permet de visuellement attirer l'attention sur le bloc de texte, avec un encadré bleu.
    Le Niveau d'exigence permet de définir si un document est obligatoire, complémentaire ou recommandé. Lorsqu'on ajoute du texte, cette option est moins pertinente. Il est donc conseillé de la laisser vide.
    En dessous de la zone de texte dans laquelle le texte est saisi, se trouve l'option Déplacer vers. Celle-ci permet de déplacer ce contenu texte dans une autre section du plan de cours (ex : matériel pédagogique, présentation du cours).
  6. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite dans l'interface.

Note : Si vous décidez de copier-coller un texte provenant d'une autre application (ex : Word), il est important d'enlever les codes qui sont importés lors de ce copier-coller et qui pourraient interférer avec la mise en page. Pour ce faire, une fois que vous avez collé le texte dans la zone de texte du plan de cours, surlignez le bloc de texte copié. Dans la barre de mise en page, juste au-dessus de la zone de texte, cliquez sur le bouton Tx. Vous aurez à formater à nouveau votre texte. Notez qu'il n'est pas garanti que cette procédure enlève tous les codes. Il est toujours préférable de saisir le texte directement dans l'application.

Modifier du texte

Pour modifier du contenu texte dans le plan de cours :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle vous voulez modifier du contenu texte (ex : présentation du cours, matériel pédagogique).
  2. Passez le curseur de votre souris sur le bloc de texte que vous souhaitez modifier et cliquez sur le bouton Modifier qui apparaît.
  3. Vous êtes alors en mesure de saisir/modifier du contenu texte dans la zone qui contient la phrase Tapez votre texte ici. Les fonctionnalités de mise en page se trouvent juste au dessus de cette zone.
  4. Vous pouvez aussi modifier les paramètres du bloc de texte (voir ci-dessus, étape 5).
  5. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite dans l'interface.

Supprimer du contenu texte

Pour supprimer du contenu texte dans le plan de cours :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle vous voulez supprimer du contenu texte (ex : présentation du cours, matériel pédagogique).    
  2. Passez le curseur de votre souris sur le bloc de texte que vous souhaitez supprimer et cliquez sur le bouton Supprimer qui apparaît.
  3. Sur l'écran de confirmation, cliquez sur Ok pour supprimer le contenu.
