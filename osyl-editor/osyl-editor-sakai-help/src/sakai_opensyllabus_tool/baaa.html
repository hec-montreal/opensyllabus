Aperçu

L'outil Plan de cours permet aux enseignants d'éditer et de publier leur plan de cours en ligne. Suite à la publication, le plan de cours devient accessible aux étudiants inscrits dans le groupe-cours. Si l'enseignant le souhaite, il peut rendre visible des éléments du plan de cours à ses étudiants seulement, à la communauté HEC Montréal (tous ceux qui ont un matricule) ou à tous les internautes sur le Web (public). L'outil plan de cours fait partie de l'environnement numérique d'apprentissage ZoneCours 2.0.

Concepts-clés

Site de cours : Dans l'environnement numérique d'apprentissage ZoneCours 2.0, un site de cours  est créé pour chaque cours ou chaque groupe-cours. Un site de cours contient par défaut certains outils, dont l'outil plan de cours. Il y a aussi des outils optionnels que l'enseignant peut décider d'ajouter au site.

Plan de cours : Le plan de cours est un des outils d'un site de cours. Présent par défaut dans tous les sites de cours, cet outil permet aux enseignants d'éditer le plan de cours et de le publier.

Types de plans de cours

Plan de cours spécifique : Les sites de cours sont identifiés selon le numéro du cours, la session et le groupe-cours (ex : 2-410-08.A2011.B01). Le plan de cours qui se trouve à l'intérieur de ce site est le plan de cours spécifique, c'est-à-dire que ce plan de cours est spécifique au groupe-cours B01. Les modifications apportées par l'enseignant au plan de cours spécifique ne se retrouvent pas dans les plans de cours d'un autre groupe (ex : groupe-cours A01) du même cours.

Plan de cours partageable : Un plan de cours partageable est automatiquement généré lors des cours coordonnées (lorsqu'il y a X groupes-cours ou plus dans un cours). Lorsque ce plan de cours est modifié et publié, les modifications apportées se retrouvent automatiquement dans tous les plans de cours spécifiques. Cette option est très utile pour établir un tronc commun à tous les  plans de cours spécifiques. Le plan de cours partageable est uniquement accessible au coordonnateur du cours. Le site de cours partageable est identifié par le numéro du cours et la session (ex : 2-410-08.A2011) sans la mention du groupe-cours.
