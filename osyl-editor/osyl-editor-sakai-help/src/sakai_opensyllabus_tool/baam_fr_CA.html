Publier un plan de cours

Vous contrôlez entièrement la publication de vos plans de cours. Il n'y a aucune publication automatique. Les étudiants ont accès au site du cours (incluant le plan de cours) dès leur inscription, donc assurez-vous de publier seulement une version que vous êtes prêt à montrer aux étudiants. Une fois qu'un plan de cours a été publié, si vous souhaitez le retirer, il est possible de le dé-publier. Il est aussi possible de publier plusieurs plans de cours simultanément.

Pour publier un plan de cours :

À l'intérieur du plan de cours, le bouton Publier se trouve parmi les boutons d'action, sur le menu horizontal supérieur, sur la droite. Cliquez sur le bouton Publier

Au besoin, consultez les sections de l'aide sur comment publier plusieurs plans de cours simultanément ou sur comment dé-publier un (ou plusieurs) plan(s) de cours.
