Éditer un plan de cours

Plusieurs types d'éléments peuvent être rajoutés au plan de cours : textes, documents, hyperliens, coordonnées, références bibliographiques, liens internes vers d'autres outils dans l'environnement numérique d'apprentissage ZoneCours 2.0.  Certains éléments peuvent être rajoutés dans plusieurs sections du plan de cours, tandis que d'autres sont spécifiques à certaines sections. Par exemple, l'élément « coordonnées » ne peut être ajouté que dans la section Coordonnées. La démarche pour ajouter un élément est la même dans toutes les sections, soit via le bouton "Ajouter".  Pour chacun des éléments, vous devez spécifier le contexte d'utilisation (niveau de diffusion) et attribuer une rubrique . 

Note : Pour les cours coordonnés, le coordonnateur a accès automatiquement au plan de cours partageable. Quand le plan de cours partageable est modifié, les modifications se retrouvent automatiquement dans les plans de cours spécifiques de tous les groupes-cours.

Pour en savoir plus sur les plans de cours partageables et spécifiques, consultez l'aide du Gestionnaire de plan de cours.
