Créer un plan de cours

Les sites des cours que vous enseignez deviennent disponibles quelques semaines avant le début de chaque session. Pour les cours qui vous sont assignés plus tard, ils apparaissent au fur et à mesure dès que PeopleSoft, le progiciel de gestion intégrée de l'École, possède cette information et la communique de façon automatisée à ZoneCours 2.0.

Au départ, le plan de cours dans chaque site de cours est « vide », c'est-à-dire qu'il contient seulement la structure du plan de cours avec les 5 sections (Présentation du cours, Coordonnées, Matériel pédagogique, Évaluations et Organisation du cours), mais ces sections n'ont pas de contenu. Pour créer le contenu, vous avez deux options : ajouter directement le contenu dans le plan de cours ou importer/copier le contenu à partir d'un autre site via le Gestionnaire de plan de cours. Ce second scénario s'applique principalement si vous avez enseigné le même cours à une session précédente, et qu'il n'y a pas eu de modifications importantes au cours. Il peut aussi s'appliquer si vous avez préparé votre plan de cours dans un site de travail créé grâce à l'outil Gestionnaire de plan de cours. Si vous importez ou copiez le contenu d'un plan de cours, tous les contenus des cinq sections sont copiés.  Par la suite, vous pouvez modifier ces contenus.
