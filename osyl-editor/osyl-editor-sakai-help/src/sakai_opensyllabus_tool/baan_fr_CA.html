Créer, ajouter, modifier et supprimer une référence bibliographique

L'enseignant peut ajouter des références bibliographiques dans le plan de cours et faire des liens directs vers les références de la bibliothèque. S'il s'agit d'une ressource qui a été publiée, il est important de la rajouter au plan de cours sous forme de référence : on ne peut ajouter directement un document pdf (ou tout autre fichier) provenant d'un texte publié sous peine d'enfreindre la loi sur les droits d'auteur. Même si vous êtes l'auteur de l'article publié, et que vous possédez le pdf de l'article, vous devez rendre l'article disponible via un lien vers la bibliothèque.

Il est possible d'insérer des références bibliographiques dans plusieurs sections du plan de cours, dont Matériel pédagogique, Évaluations et Organisation du cours (dans les séances). La démarche pour ajouter une référence bibliographique est la même dans chacune des sections.

Il est possible d'insérer des références bibliographiques vers divers types de ressources : livres, articles de périodiques et de journaux, actes de conférences, rapports du gouvernement, thèses, etc.

Le rôle de l'outil Ressources dans la gestion des références bibliographiques

Contrairement aux ajouts de documents où on peut choisir de téléverser le document directement dans le plan de cours, il faut impérativement passer par l'outil Ressources pour chercher et importer les références bibliographiques. Par la suite, vous pouvez insérer les références bibliographiques dans le plan de cours. Par défaut, il y a une liste de références intitulée Références bibliographiques du cours, qui contiendra les références que vous allez y déposer. Cependant, vous êtes en mesure de créer d'autres listes aussi, pour organiser vos références.

Les étapes de la création d'une référence bibliographique

La création d'une référence bibliographique implique d'effectuer une recherche pour trouver la référence, puis de l'importer dans l'environnement Sakai. Dans de rares cas, la référence ne peut être trouvée et il faut alors remplir un formulaire pour la créer manuellement. Chaque référence se retrouve à l'intérieur d'une liste de références (par défaut, chaque plan de cours contient une liste vide de références bibliographiques). Au moment de l'importation de la référence, les liens se font automatiquement avec les banques de données de la bibliothèque. Pour certains documents, seule une copie papier est disponible à la bibliothèque (manuels, etc).

Lorsque les références bibliographiques existent à l'intérieur de la liste de références, elles doivent être insérées dans le plan de cours pour être visible par les étudiants. La même référence peut être insérée dans plusieurs sections du plan de cours.

Les étapes du processus sont toujours les mêmes, peu importe le type de références:

  • recherche de la référence,
  • importation (ou création manuelle à l'occasion),
  • insertion dans le plan de cours.

Cependant, les étapes pour chercher des références bibliographiques et les importer dans Sakai diffèrent légèrement, selon le type de ressources.
