L'interface et les fonctionnalités de l'outil Plan de cours

Accéder au plan de cours

Chaque site de cours contient l'outil Plan de cours. Il se trouve dans le menu vertical du site de cours, à gauche dans l'interface. Cliquer sur le lien intitulé Plan de cours vous permet d'accéder à cet outil.

Le menu du cours

L'interface de l'outil Plan de cours est divisée en deux zones (rectangles). Celui à gauche, intitulé Menu du cours, contient les différentes sections du plan de cours : Présentation du cours, Coordonnées, Matériel pédagogique, Évaluations, et Organisation du cours. La section Présentation du cours permet de donner une vue globale du cours en traitant des objectifs d'apprentissages et des approches pédagogiques. La section Coordonnées permet d'afficher les coordonnées de personnes-ressources pour les étudiants : enseignant, secrétaire, stagiaire, etc. La section Matériel pédagogique permet de mettre les références de toutes les ressources utilisées . La section Évaluations permet de décrire les activités d'évaluation. Finalement, la section Organisation du cours permet de décrire le déroulement et la séquence temporelle du cours.

Ces sections se retrouvent dans tous les plans de cours. Les noms et l'ordre des sections ne sont pas modifiables afin que tous les plans de cours aient la même structure pour simplifier la navigation des étudiants. À l'intérieur de ces sections,  l'enseignant peut ajouter de nombreux types de contenus et utiliser de nombreuses options pour personnaliser son plan de cours selon les besoins du cours.

La zone d'édition du plan de cours

 En cliquant sur une des sections dans le Menu du cours (rectangle gauche), les éléments de cette section s'affichent dans le rectangle droit. Le rectangle à droite est l'espace d'édition du plan de cours : les éléments déjà ajoutés s'y trouvent et il est possible de les modifier, de les supprimer ou d'ajouter d'autres éléments. Pour l'ajout de nouveaux éléments, il faut cliquer sur le bouton Ajouter qui se trouve en haut de cette zone.

Navigation entre le Menu du cours et la zone d'édition

Pour changer la taille de ces deux zones, il suffit de cliquer dans la zone entre les deux rectangles, et par la suite de glisser la souris vers la droite ou vers la gauche. Pour afficher uniquement la zone d'édition en plein écran, cliquez sur la « flèche » entre les deux espaces. Pour réafficher la zone Menu du cours, il suffit de cliquer de nouveau sur la « flèche ».

Les boutons d'actions du Plan de cours

Les boutons d'actions qui agissent sur l'ensemble du plan de cours se trouvent au-dessus de la zone Menu du cours et de la zone d'édition. Ces boutons d'action sont : Accueil, Affichage, Enregistrer, Aperçu, Publier et Imprimer. Placé en haut à gauche, le bouton Accueil permet à tout moment de revenir à l'interface initiale du plan de cours. Dans ce cas, dans la zone d'édition à droite, les éléments de la section Présentation du cours s'affichent puisque c'est la première section du plan de cours.

Le bouton Affichage contient un menu déroulant avec ces deux options : Tout afficher et Afficher les nouveautés depuis. L'option Tout afficher permet de donner une vue globale en affichant tous les contenus de toutes les sections dans la zone d'édition à droite. L'option Afficher les nouveautés depuis : permet d'afficher les ajouts dans le plan de cours à partir d'une date choisie dans le calendrier qui s'affiche. Les sections modifiées sont surlignées en jaune dans l'espace Menu du cours (ex : Présentation du cours). En cliquant sur la section souhaitée, les éléments ajoutés depuis la date choisie sont surlignés en jaune dans la zone d'édition. Pour désactiver ce mode, cliquez sur Désactiver l'affichage des nouveautés dans le menu déroulant de l'option Affichage.

Le bouton Enregistrer permet de sauvegarder les changements apportés au plan de cours. Une sauvegarde automatique est effectuée par le logiciel toutes les 5 minutes. Lors de l'enregistrement, les modifications apportées au plan de cours ne sont pas visibles par les étudiants. Pour ce faire, il faut publier le plan de cours.

Le bouton Aperçu contient trois options dans le menu déroulant. Cette fonctionnalité vous permet une vue du plan de cours selon la perspective de différents groupes d'utilisateurs. La version des étudiants inscrits montre la vue qu'ont les étudiants du groupe-cours. La version de la communauté HEC montre la vue de ceux qui peuvent se connecter à ZoneCours 2.0 (enseignants, employés et étudiants de HEC) mais qui ne sont pas inscrits au cours. La version publique affiche la vue des internautes qui naviguent dans les plans de cours sans accès authentifié à ZoneCours 2.0. Les deux premières options peuvent être utile, par exemple, pour partager un solutionnaire avec les étudiants inscrits au cours, sans se soucier que de futurs étudiants puissent y accéder. Le partage avec la communauté HEC peut, notamment, permettre de partager des ressources de son cours avec d'autres enseignants de l'École.

Le bouton Publier permet la publication du plan de cours. Attention, la publication est immédiate etrend les modifications immédiatement accessibles aux lecteurs (dont les étudiants du cours).

Le bouton Imprimer permet d'imprimer le plan de cours. Cette fonctionnalité contient deux options dans son menu déroulant : la version en édition, la dernière version sauvegardée et la version publiée, la dernière version publiée du plan de cours.
