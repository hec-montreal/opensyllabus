Ajouter, modifier et supprimer une coordonnée

L'enseignant peut ajouter des coordonnées dans le plan de cours. Les coordonnés peuvent uniquement être ajoutées dans la section Coordonnées. Les coordonnées fréquemment affichées dans le plan de cours sont celles du coordonnateur du cours, de l'enseignant, de la secrétaire et du stagiaire, s'il y a lieu.

Ajouter une coordonnée

Pour ajouter une coordonnée dans le plan de cours :

  1. Dans le rectangle à gauche intitulé Menu du cours, cliquez sur la section Coordonnées.   
  2. Dans l'espace d'édition à droite (rectangle à droite), cliquez sur Ajouter, à la droite de l'interface.
  3. Dans le menu déroulant, choisissez Coordonnées.
  4. L'élément intitulé Coordonnée pas encore définie s'affiche à l'écran, dans le rectangle à droite.
  5. Passez le curseur de votre souris sur Coordonnée pas encore définie et cliquez sur le bouton Modifier qui apparaît suite à cette action.
  6. Dans l'interface qui s'ouvre, saisissez une coordonnée.
  7. Avec le Niveau de diffusion (en haut à droite), vous pouvez définir les groupes qui seront en mesure de visionner cette coordonnée. Pour plus d'informations, voir la page Rubriques et niveau de diffusion .
  8. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite de la fenêtre.

Modifier une coordonnée

Pour modifier une coordonnée dans le plan de cours :

  1. Dans le rectangle à gauche intitulé Menu du cours, cliquez sur la section Coordonnées.
  2. Passez le curseur de votre souris sur la coordonnée que vous souhaitez modifier et cliquez sur le bouton Modifier qui apparaît.
  3. Dans l'interface qui s'ouvre, vous êtes en mesure  de modifier la coordonnée et de définir ses paramètres.
  4. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite.

Supprimer une coordonnée

Pour supprimer une coordonnée dans le plan de cours :

  1. Dans le rectangle à gauche intitulé Menu du cours, cliquez sur la section Coordonnées.
  2. Passez le curseur de votre souris sur la coordonnée que vous souhaitez supprimer et cliquez sur le bouton Supprimer qui apparaît.
  3. Sur l'écran de confirmation, cliquez sur Ok pour supprimer le contenu.
