Rubriques et niveau de diffusion

Les rubriques

Les rubriques permettent de classifier votre contenu à l'intérieur d'une section, en attribuant un sous-titre. Les rubriques sont contextuelles et varient en fonction de la section du plan de cours. Chaque fois que vous ajoutez un contenu, vous devez sélectionner une des rubriques dans le menu déroulant. Les rubriques s'affichent dans le plan de cours seulement si vous y associez un (ou des) éléments de contenus. Elles s'affichent selon un ordre prédéterminé. Par exemple, dans la section Matériel pédagogique, l'ordre est le suivant : les contenus classés sous la rubrique «description» apparaissent en premier, suivis de ceux classés sous «objectifs» et de ceux classés sous «approche pédagogique».

L'utilisation d'un vocabulaire normalisé permet la standardisation des plans de cours. Celle-ci facilite grandement la compréhension et la recherche d'information dans un plan de cours pour les étudiants comme pour le grand public.

À l'intérieur d'une même section, les rubriques sont les mêmes pour tous les types d'éléments de contenus ajoutés (ex : texte, document, référence bibliographique, hyperlien, etc.). Plusieurs éléments (de même type ou de types différents) peuvent être classés sous la même rubrique.

Le niveau de diffusion d'un contenu

Chaque fois que vous ajoutez un élément de contenu (ex : texte, document, etc.) vous devez définir son niveau de diffusion, c'est-à-dire le groupe qui y aura accès. Il y a trois niveaux de diffusion :

Étudiants inscrits : cet élément de contenu est seulement visible par les seuls participants du site de cours (enseignant, secrétaire, étudiants inscrits au groupe-cours). Vous pouvez vérifier cette liste dans l'outil Information du site.

Communauté HEC : cet élément de contenu est visible par toutes les personnes qui possèdent un matricule HEC (employés et étudiants de l'École).

Public : cet élément de contenu est visible par tous les internautes.

Un des trois niveaux est sélectionné par défaut dans l'interface de l'ajout d'un contenu, mais l'enseignant est en mesure de le modifier.

Voici les niveaux de diffusion recommandés pour les types de contenus :

  • Étudiants inscrits : les liens vers un outil (ex : remise de travaux) et les fichiers confidentiels (solutionnaire, etc.). 
  • Communauté HEC : certains fichiers téléversés, matériel pédagogique, notes de cours, exercices, etc. 
  • Public : les textes, les liens vers un site Web, les références bibliographiques et les autres fichiers téléversés. Il est recommandé de laisser, dans toutes les sections, certaines parties (dont le texte) visibles aux internautes pour partager les plans de cours avec le grand public.
