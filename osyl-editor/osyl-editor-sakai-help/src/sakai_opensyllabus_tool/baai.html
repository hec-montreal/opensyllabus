Ajouter, modifier et supprimer un lien vers un outil

L'enseignant peut ajouter des liens vers des éléments d'autres outils de ZoneCours 2.0.  Par exemple, si l'enseignant a créé un élément pour la remise électronique d'un travail dans l'outil Remise de travaux, il peut faire un lien vers cette instance dans le plan de cours. La fonctionnalité Lien vers un outil permet de créer ce lien interne qui facilitera grandement la vie de l'étudiant. En effet, ce dernier pourra, dans la même interface où il trouvait les consignes pour son travail, remettre ce même travail en cliquant simplement sur le lien que vous aurez ainsi créé.  Il est seulement possible d'insérer ce type de lien dans les sections Évaluations et Organisation du cours du plan de cours.

Ajouter un lien vers un outil

La section ci-dessous montre les étapes pour ajouter un lien vers un outil dans le plan de cours.

  1. Dans le Menu du cours (rectangle de gauche), cliquez sur le nom de l'évaluation (ex : examen intra) ou de la partie de l'Organisation du cours à partir desquels vous souhaitez créer le lien.
  2. Dans l'espace d'édition à droite, cliquez sur Ajouter et dans le menu déroulant, choisissez Lien vers un outil.
  3. L'élément intitulé Non défini- Tapez votre texte ici s'affiche à l'écran.
  4. Passez le curseur de votre souris sur Tapez votre texte ici et cliquez sur le bouton Modifier qui apparaît.
  5. Dans la partie supérieure de l'interface, Sélectionner un outil Sakai, vous devez sélectionner l'élément de l'outil (ex : Remise du travail final) vers lequel vous voulez faire le lien. Les noms de tous les outils Sakai (ex : Tests & Quiz, Remise de travaux, etc.) dans lesquels vous avez créé des éléments sont affichées. Cliquez sur le + à côté du nom de l'outil pour afficher les titres des éléments (ex : remise pour le travail final) et cliquez sur le titre vers lequel vous souhaitez faire un lien.
  6. Le lien contenant le titre de l'élément sélectionné apparaît dans la case intitulée Outil sélectionné.
  7. Les autres paramètres permettent de définir le contexte d'utilisation du lien.
  8. En partant du haut à gauche, la première option s'intitule Rubrique et la seconde Niveau de diffusion. Pour plus d'informations sur les rubriques et sur les niveaux de diffusion, lisez la section de l'aide Rubriques et niveau de diffusion.
    Ensuite, il y a deux Options qui peuvent être activées, en cochant la case à côté de chacune. L'option Caché permet de cacher le lien suite à la publication, et de le rendre visible au moment opportun. L'option Important permet de visuellement attirer l'attention sur le lien, avec un encadré bleu.
    Le Niveau d'exigence permet de définir trois niveaux : obligatoire, complémentaire ou recommandé.
  9. Juste en dessous de ces paramètres, se trouve le seul paramètre obligatoire dans cette section, le Texte cliquable. Celui-ci constitue le libellé du lien, visible aux utilisateurs. Par défaut, il est indiqué texte cliquable dans ce champ, mais il est fortement recommandé de le changer (ex : lien vers la remise du travail final du 10 avril).
  10. En dessous de la case du Texte cliquable, se trouve une zone de texte pour saisir un commentaire au besoin.
  11. En dessous de la zone réservée aux commentaires se trouve l'option Déplacer vers. Celle-ci permet de déplacer le lien dans une autre section du plan de cours.
  12. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite de la fenêtre.

Modifier le contexte d'utilisation d'un lien vers un outil

Pour modifier le contexte d'utilisation d'un lien vers un outil :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle se trouve le lien que vous souhaitez modifier.
  2. Passez le curseur de votre souris sur le lien que vous souhaitez modifier et cliquez sur le bouton Modifier qui apparaît.
  3. Dans l'interface qui s'ouvre, vous êtes en mesure  de saisir/modifier le lien et définir/modifier ses paramètres (voir ci-dessus pour la procédure complète).
  4. Pour sauvegarder les modifications apportées, cliquez sur le bouton Ok, en bas à droite de la fenêtre.

Supprimer un lien vers un outil

Pour supprimer un lien dans le plan de cours :

  1. Dans le Menu du cours (rectangle à gauche), cliquez sur le nom de la section du plan de cours dans laquelle se trouve le lien que vous souhaitez supprimer.    
  2. Passez le curseur de votre souris sur le lien à retirer et cliquez sur le bouton Supprimer qui apparaît.
  3. Sur l'écran de confirmation, cliquez sur Ok pour supprimer le contenu.
